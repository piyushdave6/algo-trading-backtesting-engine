🚀 Algorithmic Trading Backtesting Engine

A scalable and modular algorithmic trading backtesting system designed to evaluate systematic strategies across multiple equities (NIFTY 50 universe).

📌 Overview

This project implements a structured backtesting framework capable of:

* Running strategies across multiple stocks
* Generating automated performance reports
* Applying risk-defined trading logic (TP/SL)
* Supporting modular strategy integration


⚙️ Key Features

* 📊 Multi-asset backtesting (NIFTY 50 stocks)
* 🧠 Plug-and-play strategy architecture
* 📅 Daily trade segmentation logic
* 📈 Automated Excel-based reporting
* ⚡ Vectorized + rule-based execution



🏗️ Project Structure


algo-trading-backtesting-engine/
│
├── strategies/        # Strategy logic (MACD, etc.)
├── backtester/        # Core backtesting engine
├── data/              # Input market data
├── reports/           # Output reports (Excel)
├── utils/             # Helper functions
└── main.py            # Execution entry point

▶️ How to Run

bash
python main.py
